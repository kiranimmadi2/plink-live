# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-14

### Added
- Initial release of Supper app
- AI-powered intent matching system
- Real-time profile matching based on user prompts
- Voice calling functionality using WebRTC
- Firebase authentication with Google Sign-In
- Chat messaging system
- Location-based matching (using Geolocator)
- Price-based matching for marketplace items
- Profile management system
- Push notifications for matches and messages
- Clarification question system for ambiguous intents

### Known Issues
- Voice call profile display: Users see their own profile instead of the other person's profile during calls

## Version Format
- Format: MAJOR.MINOR.PATCH+BUILD
- Example: 1.0.0+1
  - MAJOR: Breaking changes
  - MINOR: New features (backwards compatible)
  - PATCH: Bug fixes
  - BUILD: Internal build number
